About

The goal of this website is to be as accessible as possible for all users and to serve as an educational tool to help people learn more about web accessibility. This includes ensuring a user-friendly experience for individuals using screen readers, keyboard navigation, and other assistive technologies. This project is part of a course focused on learning and applying key web accessibility principles. We have implemented features such as semantic HTML, keyboard navigation, ARIA roles for screen reader compatibility, and attention to color contrast.

For everyone. Built by Ida, Oskar and Tilde.
