How this Quiz is accessible

  1. A "Skip to main content" link lets users bypass navigation and go directly to the quiz.
  2. The quiz is fully keyboard-accessible, allowing users to navigate and select answers without a mouse.
  3. A clean and simple layout ensures an easy-to-read experience with minimal distractions.
  4. ARIA labels enhance accessibility for screen readers.
  5. Semantic HTML improves structure and readability for both users and assistive technologies.
  6. A dark mode option provides a high-contrast black background with white text for a more comfortable viewing experience.
  7. A reduce motion option allows users to disable animations and sliding effects in the quiz, providing a more comfortable experience for anyone sensitive to motion.
